# Kit modular de personajes SIEP

Todas las hojas PNG miden **192×288 px**, con una cuadrícula **3×3** y frames de **64×96 px**.

Orden de capas: `hair back → body → face → hair front`.

## Cuerpos

`body/body_{female|male}_{purple|blue|green|burgundy|gray}.png`

## Cabello

`hair/hair_{short|medium|long|tied}_{black|brown|blonde|red|gray}_{back|front}.png`

## Expresiones

- `face_neutral.png`
- `face_calm.png`
- `face_worried.png`

La selección de color de ropa se realiza cambiando el archivo corporal utilizado por el avatar.
